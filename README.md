# 座席 — 座席決めアプリ

飲み会・席替え・貸切バスなど、あらゆるシーンで使える座席決めWebアプリです。
自由なレイアウトで座席を配置し、ランダムに席を割り当てられます。

**URL:** https://mikotommsh.github.io/zaseki/

---

## 使い方

### 1. 座席を配置する

キャンバス（右側の白い領域）をクリックすると座席が追加されます。
座席はドラッグして自由に移動できます。

### 2. 出席者を登録する

左パネルの入力欄に名前を入力し「＋」ボタンまたは Enter キーを押します。
出席者は何人でも追加できます。

### 3. 席を決める

「🔀 席決め」ボタンを押すと、出席者がランダムに座席へ割り当てられます。
ボタンを押すたびに再シャッフルされます。

### 4. 手動で割り当てる

キャンバスで座席をクリックして選択状態にした後、左パネルの出席者名をクリックすると、その人をその席に割り当てられます。

### 5. ピン留め

座席の鍵アイコン（🔒）をクリックするとその席がピン留めされます。
ピン留めされた座席は「席決め」ボタンを押しても割り当てが変わりません。
再度クリックで解除できます。

### 6. ラベルを編集する

座席またはオブジェクトをダブルクリックするとラベルを編集できます。
「A-1」「窓際」「VIP」など任意のテキストを入力できます。

### 7. オブジェクトを配置する

「＋ オブジェクト配置」ボタンをクリックすると配置モードになります。
キャンバスをクリックすると「オブジェクト」が配置されます。
ダブルクリックで「入口」「演台」「ホワイトボード」などのラベルに変更することで、部屋の目印として使えます。
ESC キーまたは「✕ 配置モード解除」ボタンで配置モードを終了します。

### 8. 削除する

座席またはオブジェクトを右クリックすると削除されます。
出席者は左パネルの「×」ボタンで削除できます。

### 9. 画像として書き出す

「🖼 書き出し」ボタンを押すと、現在のキャンバスをPNG画像としてダウンロードできます。

### 10. URLで共有する

「🔗 共有」ボタンを押すと、現在の状態（座席・出席者・オブジェクト・割り当て）をすべて含んだURLがクリップボードにコピーされます。
そのURLを送ることで、同じ状態を別の端末・別のブラウザで開けます。

---

## 操作まとめ

| 操作 | 内容 |
|------|------|
| キャンバスをクリック | 座席を追加 |
| 座席をドラッグ | 座席を移動 |
| 座席をダブルクリック | ラベルを編集 |
| 座席を右クリック | 座席を削除 |
| 座席の🔒アイコン | ピン留めのON/OFF |
| 出席者の×ボタン | 出席者を削除 |
| ESCキー | 配置モードを解除 |

---

## 技術情報

- フロントエンドのみで動作（サーバー不要）
- ページリロードで状態はリセットされます（保存にはURL共有を使用）
- React 18 + TypeScript + Vite
- GitHub Pages でホスト
